# design-misc / Портфолио макетов

Добро пожаловать!  
Здесь собраны макеты для печати (визитки, постеры и т.д.) и цифровых продуктов (сайты, лендинги и т.д.).

## 📂 Структура репозитория

📁 **posters/** — Постеры и плакаты
